# Bayesian Data Analysis Portfolio

A collection of mini-projects and practical assignments completed during the Bayesian Data Analysis course.

## 🛠 Technologies & Tools
The projects are implemented using Python and standard scientific computing libraries:
* **Python 3**
* **NumPy** & **SciPy** (for statistical distributions and math operations)
* **Matplotlib** (for plotting probabilities and data visualization)
* **PyMC** & **ArviZ** (for MCMC sampling and posterior analysis)
* **Jupyter Notebook**

## 📂 Projects

* `assignment-1/` — Estimating probabilities and applying Bayes' theorem to the 100-sided die problem.
* `assignment-2/` — Estimating the position of a lighthouse using Bayesian inference and the Cauchy distribution (1D, grid method).
* `assignment-3/` — Estimating the position and height of a lighthouse using 2D joint posterior inference (grid method).
* `assignment-4/` — Estimating the lighthouse position and height using PyMC and MCMC sampling.
* *(More projects will be added here as the course progresses)*
## 💡 About This Repository
This repository serves as a personal showcase of solving probability and statistics problems using the Bayesian approach. Instead of theoretical lectures, this workspace is dedicated purely to code implementations, problem-solving, and data visualization.
